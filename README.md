# AIWD_LAB4
Artificial Intelligence in Web Development Laboratory 4

Description

This experiment demonstrates the use of JavaScript Document Object Model (DOM) methods and events to create an interactive web application. The webpage allows users to convert currency values dynamically using a dropdown selection. DOM methods such as getElementById, getElementsByClassName, getElementsByTagName, and querySelectorAll are used to access and modify HTML elements. JavaScript events such as onload, onclick, onchange, and onkeyup are used to handle user interactions. The application performs client-side calculations and updates results dynamically without refreshing the page.
